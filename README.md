# Swavalambi-ai Web App

Skills to Self Reliance - AWS Amplify Gen 2 Application

## Setup

```bash
# Install dependencies
npm install

# Start local development with cloud sandbox
npx ampx sandbox

# In another terminal, start dev server
npm run dev
```

## Deploy

```bash
# Deploy to AWS
npx ampx deploy --branch main
```

## Features

- Phone OTP authentication
- User registration with Name, Age, Mobile
- User dashboard
- Logout functionality

## Tech Stack

- React + TypeScript
- Vite
- AWS Amplify Gen 2
- AWS Cognito (Phone OTP)
